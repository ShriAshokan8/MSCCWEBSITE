üî¨ Our Making Process

Every project follows a maker's mindset:

  1. Ideate: Brainstorm ideas and sketch concepts
  2. Prototype: Build rough versions to test feasibility
  3. Experiment: Try different materials, colors, and techniques
  4. Test & Refine: Evaluate what works, improve what doesn't
  5. Produce: Create final versions with attention to quality
  6. Reflect: Learn from each project to inform the next

  • Engineering: Design thinking, prototyping, problem-solving, iteration
