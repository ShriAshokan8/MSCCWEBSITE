CONFIGURATION REQUIRED
---------------------
Users must configure before deployment:

1. FIREBASE:
   - Create Firebase project
   - Enable Authentication (Email/Password)
   - Enable Firestore Database
   - Update config in nexus/js/firebase.js
   - Deploy functions: firebase deploy --only functions
   - Apply Firestore security rules (see nexus/README.md)

2. SUPABASE:
   - Create Supabase project
   - Create storage bucket: msc-nexus
   - Update config in nexus/js/supabase.js
   - Apply storage policies (see nexus/README.md)
