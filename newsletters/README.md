# Newsletters

This folder contains newsletters and related documents for the MSC Initiative.

## Current Files

- Sample-Newsletter.pdf (placeholder) - Example newsletter demonstrating the format and content structure

## How to Add New Newsletters

1. Upload newsletter files (preferably PDF format) to this directory
2. Update the newsletters.html page to include download links to new files
3. Follow the naming convention: `YYYY-MM-Newsletter-Title.pdf`

## Guidelines

- Newsletters should be in PDF format for consistency
- File names should be descriptive and include the date
- Keep file sizes reasonable for web distribution
